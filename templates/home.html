Customer Churn Prediction

About the Project

This project helps telecom companies predict whether a customer will churn (leave) or stay. It collects customer data and uses machine learning to provide predictions based on behavior and usage patterns.

Objectives

  • Collect relevant customer data through a user-friendly form
  • Analyze the data using predictive models
  • Classify each customer as likely to churn or not churn

How It Works

  1. User enters customer details in the form
  2. Form data is sent to a backend model
  3. The model returns a prediction: Churn or Not Churn

Technologies Used

  • HTML, CSS, JavaScript for frontend
  • Python and Machine Learning models
  • Flask for backend integration

Who Can Use This?

Business analysts, telecom operators, and developers looking to understand and prevent customer churn can use this tool.
